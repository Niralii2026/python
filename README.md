# 🔄 Python Swap Examples

This repository demonstrates different ways to swap values in Python using simple and beginner-friendly examples.

---

## 📌 Features

* Swap two variables using **tuple unpacking (Pythonic way)**
* Swap using a **temporary variable**
* Swap elements inside a **list**

---

## 🧠 Concepts Covered

* Variable assignment
* Tuple unpacking
* List manipulation
* Function usage in Python

---

## 🚀 Why Use Tuple Swapping?

Python provides a clean and efficient way to swap values without needing a temporary variable:

```python
a, b = b, a
```

This is the most recommended and commonly used approach.
